Want to put this site online?

Two common paths: keep this as a static HTML site, or install WordPress for a full CMS.

Option A — Static Hosting (for this HTML file)

  1. Buy a domain from your preferred registrar.
  2. Buy hosting (shared or cloud) from your hosting provider.
  3. Upload this index.html and your logo.png to the hosting file manager (or via FTP).
  4. Point your domain’s DNS to your hosting (A record / nameservers).

Generic help: General static hosting guide

Option B — WordPress (CMS)

  1. Buy a domain and hosting that supports WordPress (1-click or manual install).
  2. Use the host’s “Auto-Install WordPress” or follow a general WordPress install guide.
  3. Pick a theme, add pages (About, Programs, Events, Gallery, Join, Contact).
  4. Upload your club logo to “Site Identity” and set brand colors (monochrome to match your emblem).
